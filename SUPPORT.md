# Support

## For Bloccpay customers and users

- **Product help:** support@bloccpay.com
- **Website:** https://www.bloccpay.com

## For security issues

Do not use this channel for security reports. See [SECURITY.md](./SECURITY.md).

## For code / issue tracker

If you're looking at one of our repositories:

- **Bug reports** — open an issue in the affected repository using the "Bug" template.
- **Feature ideas** — open an issue using the "Feature" template. Not every idea will be built; we prioritize against roadmap.
- **Questions about how the code works** — open a "Question / Discussion" issue if it needs a durable record.
